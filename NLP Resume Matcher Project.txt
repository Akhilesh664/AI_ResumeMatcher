рџ”№ TF-IDF broken into parts
1пёЏвѓЈ Term Frequency (TF)

How often does a word appear in a document?

Example (resume):

java java spring boot
Word	Count
java	2
spring	1
boot	1

More occurrences в†’ more importance within that document.

2пёЏвѓЈ Inverse Document Frequency (IDF)

How rare is the word across ALL documents?

java appears in many resumes в†’ less special

kubernetes appears in few resumes в†’ very special

Formula (conceptually):

IDF(word) = log(total documents / documents containing word)

So:

Rare words в†’ higher score

Common words в†’ lower score

3пёЏвѓЈ TF Г— IDF = TF-IDF

Final weight:

TF-IDF = TF * IDF

So:

Frequent + rare = high weight

Frequent + common = low weight

рџ”№ What problem TF-IDF solves (very important)

Without TF-IDF:

Every word has equal weight

вЂњjavaвЂќ == вЂњtheвЂќ (bad!)

With TF-IDF:

Skills dominate

Noise disappears

Matching improves dramatically
